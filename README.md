## 📌 About

This is the official website invitation for **INNOVISION 2026** — the Annual Student Project Showcase and Innovation Exhibition organized by the **Department of Information Technology, Advanced Technological Institute (ATI), Labuduwa, SLIATE.**

The event was successfully held on **21st July 2026**, themed **"Vision Beyond Innovation"**, bringing together IT students, academic staff, and industry professionals under one roof.

---

## ✨ Features

- 🌌 **Particle Constellation Background** — Interactive canvas-based star field that reacts to mouse movement
- ⌨️ **Typing Animation Hero** — Cycles through event taglines with a cursor effect
- ⏱️ **Live Countdown Timer** — Ticking down to the event date in real time
- 📊 **Animated Stat Counters** — Numbers count up when scrolled into view
- 🗂️ **Competition Categories** — All 5 tracks including the new AI-Based Website category
- 📅 **Scroll-Triggered Timeline** — Judging process phases animate in as you scroll
- 🃏 **3D Flip Cards** — Hover to reveal the benefit on each "Why Attend" card
- 🏆 **Awards Section** — Gold, Silver, and Bronze placement cards with glow effects
- 📍 **Venue Section** — Embedded map with glassmorphism info card
- 💌 **VIP Invitation Card** — Wax seal stamp animation with golden particle border
- 💰 **Budget Breakdown** — Transparent event financials display
- 📱 **Fully Responsive** — Optimised for desktop, tablet, and mobile

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/dimuthushalinda/innovision-2026.git

# Navigate into the project
cd innovision-2026

# Open in browser
open index.html
```

No dependencies. No frameworks. No build step required — just open and run.
